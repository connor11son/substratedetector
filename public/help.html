Substrate Detection Module Help

About:

This module takes in an input video (.mp4) and returns the predicted substrate for each individual frame.

How to use:

  1. Press Select Image and navigate to the video you would like to predict the substrate for.

  2. Press Run and wait for the module to finish executing (may take a while for larger videos).

  3. Click on the link to the output file in the results box.

  4. Once viewing the output for the csv in the BisQue csv viewer, click on Download on the top of the screen and download a specified format (choose "Original" to download only the csv).
